# STGNN Crowd Analysis

A **Spatio-Temporal Graph Neural Network (STGNN)** framework for crowd analysis and anomaly detection.

##  Project Structure
- `src/models/` → STGNN model architecture  
- `src/preprocess/` → graph building, dataloaders  
- `src/train.py` → training logic  
- `src/inference.py` → inference / validation  
- `data/` → raw and processed datasets (ignored in git)  
- `outputs/` → checkpoints, logs, and visualizations
